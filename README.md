# Diet Balancer Frontend (Vite + React + Tailwind)

## Run locally

- npm install
- npm run dev

## Features

- Dashboard with nutrient chart and cards
- Food Logger to add foods with nutrient amounts
- Recommendations based on daily intake vs reference goals
- Admin Panel to edit reference values and thresholds

## Notes

- Data persisted in localStorage via src/services/mockApi.js
- Reference values seeded from src/data/nutrientReference.json
- Charts powered by Recharts
- Styling with TailwindCSS
